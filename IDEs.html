Instalar y configurar IDEs

Los IDEs proporcionan herramientas y funcionalidades para ayudar a los programadores a escribir, depurar y mantener código de manera más eficiente. Integran editor de código, compilador o intérprete y depurador en una sola interfaz, centralizando las tareas más comunes.

¿Qué son los módulos?

Los módulos son componentes o secciones específicas que permiten que el entorno ofrezca diferentes funcionalidades. También se conocen como extensiones o plugins.

IntelliJ IDEA

  • Busca en tu navegador: "IntelliJ IDEA download JetBrains".
  • Entra al sitio oficial de JetBrains.
  • Selecciona la versión Community (gratuita) o Ultimate (de pago).
  • Descarga el instalador para Windows (.exe).
  • Abre el instalador y sigue los pasos.
  • Opcionalmente activa Add to PATH y asocia extensiones como .java o .kt.
  • Haz clic en Install y abre IntelliJ.
  • En el primer inicio, importa configuraciones anteriores o usa las predeterminadas.

Instalación de módulos (plugins) en IntelliJ IDEA

  • Abre IntelliJ IDEA.
  • En el panel izquierdo, entra a Plugins.
  • Selecciona el plugin deseado, por ejemplo "JetBrains AI Assistant".
  • Haz clic en Install.

Este plugin añade un asistente con IA integrado en el IDE para ayudar en tareas de programación.

Eclipse

  • Busca en tu navegador: "Eclipse IDE download".
  • Entra al sitio oficial de Eclipse Foundation.
  • Descarga el instalador correspondiente a tu sistema.
  • Ejecuta el instalador y selecciona la distribución (Java, C/C++, PHP, etc.).
  • Selecciona carpeta de instalación y haz clic en Install.
  • Al abrir Eclipse por primera vez, selecciona un workspace.

Instalación de módulos (plugins) en Eclipse

  • Abre Eclipse.
  • Ve a Help → Eclipse Marketplace.
  • Busca el plugin deseado, por ejemplo "GitHub Copilot".
  • Haz clic en Install.

Visual Studio Code

  • Busca en tu navegador: "Visual Studio Code download".
  • Entra al sitio oficial de Microsoft.
  • Descarga la versión Windows x64 System Installer.
  • Ejecuta el instalador, acepta la licencia y marca las opciones recomendadas (PATH).
  • Instala y abre Visual Studio Code.

Extensiones útiles en VS Code

  • Auto Rename Tag: renombra automáticamente etiquetas HTML/XML.
  • Live Server: despliega un servidor de desarrollo con recarga en tiempo real.
  • Error Lens: muestra errores y advertencias directamente en el editor.

Para eliminar módulos, basta con ir a Extensiones, buscar el módulo y pulsar "Uninstall".

Personalización en VS Code

  • Cambiar tema de color desde la Command Palette → Color Theme.
  • Modificar fuente y tamaño desde File → Preferences → Settings → Font.

NetBeans

En NetBeans puedes personalizar fuentes y colores desde Tools → Options → Fonts & Colors, además de instalar plugins desde el gestor de extensiones.

IntelliJ IDEA (personalización)

Desde File → Settings → Appearance & Behavior puedes cambiar el tema principal y personalizar menús y toolbars.

Eclipse (personalización)

Desde Window → Preferences → General → Appearance puedes cambiar el tema principal y ajustar colores y fuentes.
