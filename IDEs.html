Instalación y Configuración de los IDEs Más Usados

En esta guía se explican los pasos para instalar y configurar los IDEs más utilizados tanto de forma profesional como educativa. También se incluyen configuraciones útiles, instalación de plugins y personalización.

¿Qué son los módulos o plugins?

Los módulos o plugins son componentes adicionales que amplían la funcionalidad de un IDE. Permiten añadir herramientas, soporte para nuevos lenguajes, asistentes, mejoras visuales y mucho más.

Instalación de IntelliJ IDEA

Pasos para instalar IntelliJ IDEA en Windows:

  • Buscar en el navegador “IntelliJ IDEA download JetBrains”.
  • Acceder al sitio oficial de JetBrains.
  • Seleccionar la versión Community o Ultimate.
  • Descargar el archivo instalador .exe.
  • Abrir el instalador y configurar accesos directos y opciones como agregar al PATH.
  • Asociar extensiones como .java o .kt si se desea.
  • Finalizar la instalación y abrir IntelliJ.
  • Elegir entre importar configuraciones previas o usar las predeterminadas.

Instalar módulos de interés (plugins)

  • Abrir IntelliJ IDEA.
  • Ir a Plugins en el panel izquierdo.
  • Buscar el plugin deseado.
  • Seleccionar Install.
  • Reiniciar IntelliJ si es necesario.

Instalación de Eclipse

  • Buscar “Eclipse IDE download”.
  • Entrar al sitio oficial de Eclipse Foundation.
  • Descargar el archivo “eclipse-inst-win64.exe”.
  • Elegir la distribución: Java Developers, C/C++, etc.
  • Seleccionar carpeta de instalación.
  • Finalizar instalación y abrir Eclipse.
  • Seleccionar un workspace en el primer inicio.

Instalar módulos de interés (plugins)

  • Abrir Eclipse.
  • Ir a Help > Eclipse Marketplace.
  • Buscar el plugin deseado (por ejemplo GitHub Copilot).
  • Seleccionar Install.

Instalación de Visual Studio Code

  • Buscar “Visual Studio Code download”.
  • Entrar al sitio oficial de Microsoft.
  • Descargar la versión Windows x64.
  • Ejecutar el instalador .exe.
  • Aceptar la licencia.
  • Seleccionar opciones recomendadas como agregar al PATH.

Instalación de extensiones útiles

Auto Rename Tag

Actualiza automáticamente la etiqueta de cierre al renombrar la etiqueta de apertura.

Live Server

Permite visualizar cambios en tiempo real en el navegador.

Error Lens

Muestra errores directamente en el editor junto con una descripción.

Las extensiones se instalan desde el apartado Extensiones en VS Code.

Eliminar módulos en VS Code

Ir a Extensiones, buscar el módulo instalado y seleccionar Uninstall.

Personalización: Tema de color

  • Abrir la Command Palette.
  • Escribir “Color Theme”.
  • Seleccionar un tema predefinido.

Modificar fuente y tamaño

  • Abrir Settings.
  • Buscar “Font”.
  • Modificar tipo, tamaño y estilo.

Personalizar atajos de teclado

  • Abrir Command Palette.
  • Buscar “Open Keyboard Shortcuts (JSON)”.
  • Editar el archivo keybindings.json.

Configuración de actualizaciones automáticas

En Settings buscar “update” y configurar:

  • update.channel: canal de actualizaciones.
  • update.mode: automática o manual.

Instalación de NetBeans

  • Buscar “Apache NetBeans download”.
  • Entrar al sitio oficial.
  • Descargar instalador Windows (.exe / .msi) o archivo ZIP.
  • Si es ZIP, descomprimir y ejecutar desde la carpeta /bin.

Configuración del sistema de actualización

  • Abrir NetBeans.
  • Ir a Tools > Plugins.
  • Entrar en pestaña Settings.
  • Activar o desactivar repositorios.
  • Añadir un repositorio con Add si es necesario.

Instalar plugins en NetBeans

  • Abrir Plugin Manager.
  • Entrar a Available Plugins.
  • Seleccionar plugins deseados.
  • Instalar y reiniciar NetBeans.

Generar un ejecutable en NetBeans

  • Seleccionar el proyecto.
  • Usar Clean and Build Project.
  • Buscar la carpeta dist generada.
  • Abrir terminal desde esa carpeta y ejecutar el archivo generado.

Exportar de NetBeans e importar en Eclipse

  1. Abrir File > Export Project > To ZIP.
  2. Seleccionar proyecto y exportar.
  3. Abrir Eclipse.
  4. Ir a File > Import.
  5. Seleccionar Existing Projects into Workspace.
  6. Importar desde ZIP o carpeta descomprimida.
